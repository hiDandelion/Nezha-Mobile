## 什么是Nezha Mobile

Nezha Mobile是一个非官方的Nezha Dashboard客户端。

你可以从App Store下载Nezha Mobile。

[![3e7fb65e-a6d2-4011-bbee-0a9fad60e6e0](https://github.com/user-attachments/assets/3e7fb65e-a6d2-4011-bbee-0a9fad60e6e0)](https://apps.apple.com/app/id6596729064)

或从源代码构建。

对于任何与Nezha Dashboard或Nezha Agent相关的问题，请见[Nezha Wiki](https://nezha.wiki)。
